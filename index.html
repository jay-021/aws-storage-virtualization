AWS Storage Virtualization Lab

Overview

This project demonstrates the use of various AWS storage services, including:

  • Amazon S3: Used for storing static assets like images and web files.
  • Amazon Elastic File System (EFS): A scalable file system for EC2 instances.
  • Amazon DynamoDB: A NoSQL database used for fast and predictable data access.

Why SimpleDB is Not Included

Amazon SimpleDB was not included in this project because it is a paid service and does not have a free-tier option. To keep this lab within the free-tier limits, it was excluded.

Architecture

The resources were created using an AWS CloudFormation template that automates the deployment of the following:

  • An S3 bucket named "jay-storage-virtualization-s3-bucket"
  • An EFS file system named "Jay-StorageVirtualizationEFS"
  • A DynamoDB table named "Jay-StorageVirtualizationDynamoDB"

CloudFormation Template

The CloudFormation template used for this lab is as follows:

            AWSTemplateFormatVersion: "2010-09-09" Description: AWS Storage Virtualization Lab by Jay

            Resources: JayS3Bucket: Type: "AWS::S3::Bucket" Properties: BucketName: "jay-storage-virtualization-s3-bucket" Tags: - Key: "Owner" Value: "Jay"
            
            JayEFSFileSystem: Type: "AWS::EFS::FileSystem" Properties: FileSystemTags: - Key: "Name" Value: "Jay-StorageVirtualizationEFS" - Key: "Owner" Value: "Jay"
            
            JayDynamoDBTable: Type: "AWS::DynamoDB::Table" Properties: TableName: "Jay-StorageVirtualizationDynamoDB" AttributeDefinitions: - AttributeName: "ID" AttributeType: "S" KeySchema: - AttributeName: "ID" KeyType: "HASH" ProvisionedThroughput: ReadCapacityUnits: 5 WriteCapacityUnits: 5 Tags: - Key: "Owner" Value: "Jay"
            
            Outputs: JayS3BucketOutput: Value: !Ref JayS3Bucket Description: "Jay's S3 Bucket Name"
            
            JayEFSOutput: Value: !Ref JayEFSFileSystem Description: "Jay's EFS File System ID"
            
            JayDynamoDBTableOutput: Value: !Ref JayDynamoDBTable Description: "Jay's DynamoDB Table Name"
